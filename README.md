# Vishlex - Translation Quality Evaluation

Features
 - Translate text using the DeepL API
 - Customize the target language for translation
 - Return translation results as JSON
    
Requirements
 - Python 3.7+
 - Flask
 - Requests
 - A DeepL API key (you'll need to sign up for a DeepL account and get an API key)
